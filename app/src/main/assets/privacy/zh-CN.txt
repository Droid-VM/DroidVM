DroidVM 隐私协议
更新日期： 2026年4月8日
生效日期： 2026年4月8日

感谢你使用 DroidVM（以下简称「本应用」）。本应用在安卓设备上运行虚拟机。
本隐私协议旨在向你说明：在特定功能中，本应用如何收集、使用和保护你的信息。

一、信息的收集与使用
本应用默认不会收集任何个人信息。仅当你主动同意以下协议后，本应用才会在特定功能（镜像下载索引、更新检查）中向 API 服务器（https://api.classfun.cn/）发送最低限度的技术信息：
IP 地址：用于建立网络连接及服务器访问日志记录。
UserAgent：HTTP要求的用于标识客户端类型，确保 API 兼容性。包括应用程序版本，Android版本，设备型号，设备SoC型号

服务器处理方式：
上述信息仅出现在服务器访问日志中。
服务器不会主动存储这些信息用于分析、画像或任何其他用途。
日志文件会按常规运维策略自动轮转覆盖。

二、信息的存储与共享
不存储： 除瞬时访问日志外，服务器不保存你的 IP 地址或 UserAgent。
不共享： 本应用及服务器不会向任何第三方出售、交换或转让上述信息。
不关联： 上述信息不会被关联到你的设备标识、账号或其他可识别身份的信息。

三、你的选择
同意协议： 你可以正常使用镜像下载索引、更新检查等依赖 API 的功能。
拒绝协议： 本应用不会调用任何 API，也不会发送任何信息到服务器。但依赖 API 的功能（镜像下载索引、更新检查）将不可用，其他本地虚拟化功能不受影响。
你可以在首次使用相关功能时选择是否同意，也可以在应用设置中随时更改授权状态。

四、数据安全
由于本应用仅收集瞬时日志中的 IP 和 UserAgent，且不作持久化存储，我们无法「删除」已自动轮转覆盖的日志。
API 服务器采用常规安全措施（如 HTTPS、访问控制）以防止日志被未授权访问。

五、未成年人保护
本应用不面向 14 周岁以下未成年人。如你为 14 周岁以下，请勿同意本协议或使用依赖 API 的功能。

六、协议变更
如本隐私协议发生重大变更（如增加信息收集类型），我们会通过应用内更新提示的方式通知你，并再次征求同意。
微小修订（如措辞优化、服务器地址更新）将直接更新本页面。
